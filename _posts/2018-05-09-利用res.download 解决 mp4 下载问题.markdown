查找了很多资料,发现 Nodejs 的 res.download 专门为此设计. 如果有类似需求可以用来试试. 

语法如下:

 
```JavaScript
 res.download('/xxx.pdf','dl.pdf',function(err){
	if(err){
	//如何如何

	}

 	})
```
功能不错,缺点貌似是 可下载的文件必须是 本服务器上的文件,如果是外面的文件貌似不可以.
